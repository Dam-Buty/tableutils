tableutils

A jQuery plugin for HTML tables. Fix table headers with horizontal scrolling support, add master checkbox, add sorting, filtering, client-side and server-side numeric, alphabetic and alphanumeric pagination, add new records to the table dynamically, delete, edit existing rows in the table, add custom buttons related to your table's functionality and show stylish messages about your table.

View project onGitHub

About the tableutils plugin

There are a lot of jQuery plugins available for doing things like fixing the table header, filtering/searching the table contents, sorting the table, paginating it on client-side, paginating it on server-side, adding new records to a table dynamically, editing the records on the fly, deleting records from the table, etc.

However, not many plugins have all these functionality. So why not use a single plugin to do all these things instead of using a number of plugins. This is exactly what tableutils is. It includes all these functionality, so that you get everything in one little plugin.

Features

  • Fix table header
  • Scroll the fixed header horizontally with the table body for larger tables
  • Search/Filter data in the table
  • Sort data in the table.
  • Add a Checkbox to select/de-select all rows
  • Select rows by clicking on them
  • Client-side JavaScript pagination
  • Server-side Pagination
  • Supports numeric, alphabetic and alphanumeric pagination
  • Add new records to the table dynamically
  • Update records in the table
  • Delete records from the table
  • Add buttons to your table to perform related operations
  • See information about sorting, filtering, pagination, etc in a stylish way with a custom message queue

Installation

Like every other jQuery plugin out there, installation is super easy.

  1. Add jQuery libraries to your page. 

    <script type="text/javascript" src="js/jquery.js"></script>
<script type="text/javascript" src="js/jqueryUI.js"></script>

  2. Add tableutils script file to your page. 

    <script type="text/javascript" src="js/tableutils.js"></script>

  3. Add your favourite jquery UI theme css to your page.

    <link rel="stylesheet" type="text/css" href="css/flick/jqueryUI.css" />

  4. Add tableutils css file to your page.

    <link rel="stylesheet" type="text/css" href="css/tableutils.css" />

And its done.!! That was easy right ? Now, go ahead and use the plugin to create incredibly smart tables.

How to use the plugin

Create the HTML for table. You don't have to do anything special here. But there are a few things you should know:

  1. Specify the width for the table in pixels.

  2. Specify the width for each individual column in the table (column width need not be in pixels i.e. you can use percentage width for columns). The widths for the headers and body must match for the plugin to work properly.

  3. Specify thead and tbody section in the table. thead will hold the table header and tbody will be the body of the table.

Apart from this nothing more needs to be special about the table. You see, everything here is just a basic table in HTML. See an example below: 

<table id="mySimpleTable" width="800px">
    <thead>
        <tr>
            <th width="20%">First Name</th>
            <th width="20%">Last Name</th>
            <th width="10%">Department</th>
            <th width="10%">Age</th>
            <th width="40%">Address</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td width="20%">Pratik</td>
            <td width="20%">Kolge</td>
            <td width="10%">IT</td>
            <td width="10%">23</td>
            <td width="40%">Pune, India</td>
        </tr>
        ... 
    </tbody>
</table>

And then, script the following code: 

<script type="text/javascript">
    $(document).ready(function() {
        $('#mySimpleTable').tableutils( options );
    });
</script>

And there you go, your very simple table transforms into something more lively. Go ahead and explore the awesome features of this plugin. I am sure you would love it.

A note

When I first started to write this code, I never thought that I will expand it so much. My aim was to just fix the table header since i had seen so many different methods for fixing the headers. And i wanted a simple method that isn't implemented using any CSS to avoid cross-browser irregularities. But over the time, many things were added to the code.
